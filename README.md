# react-strict-dom-turborepo

### packages/ui

```shell
~$ cd packages/ui
~$ yarn
~$ yarn build or dev
```

### web

```shell
~$ cd apps/web
~$ yarn
~$ yarn dev
```

### android

```shell
~$ cd apps/native
~$ yarn
~$ yarn android
```

### iOS

```shell
~$ cd apps/native
~$ cd ios
~$ pod install
~$ -> xcode...
```
